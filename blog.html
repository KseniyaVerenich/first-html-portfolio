First week, first post!

March 1, 2019 by Kseniya


Wow! The first week of bootcamp has come to a close, and it was quite the doozy!

First week topics included HTML, CSS, and Bootstrap. Being fairly familiar with HTML and CSS,

I felt comfortable with diving right in, however, I had no idea Bootstrap even existed!

Quickly building responsive, mobile-first projects on the web is made easy with Bootstrap,

and was a very exciting tool to learn how to use! Moving forward, I would definitely like to

continue to develop my skills in html/css/bootstrap, and learn how to create more interactive

and functional web pages.




Second week, second post!

March 8, 2019 by Kseniya


Diving into javascript this week, I was initially really excited to get started.

However, things became difficult rather quickly. Our assigned challenges/exercises tested my patience

and confidence. By Tuesday evening, I was ready to quit the program! Every new bit of information

seemed daunting, and I wasn't convinced that I'd ever be able to get a good grasp on it. On Wednesday, though,

things quickly turned around! We worked in pairs to complete our assigned "weather app" exercise, and I

had more than one "Aha!" moment. The relationship between javascript and HTML clicked, and I was able to better

understand how to use javascript to code more dynamically. I did find, however, that a lot of time was spent

trying to find minor syntax errors, which is something I would definitely categorize under "things that I hate"

as a new programmer. I am definitely a bit anxious about starting Vue next week. My fingers are crossed that the upcoming

week sees more wins than fails!
